# Project MyShop 2025 - Documentation

Tài liệu hướng dẫn và giải thích kỹ thuật cho đồ án môn học "Lập trình trên môi trường Windows".

## Mục lục Tài liệu

1.  **[Architecture & Design](Architecture_Design.md)**
    *   Tổng quan kiến trúc MVVM.
    *   Cấu trúc thư mục dự án.
    *   Dependency Injection (DI).

2.  **[Key Features Technical Deep Dive](Key_Features_Technical.md)**
    *   Giải thích chi tiết kỹ thuật các tính năng phức tạp (Import Excel, AutoSave, Reports, Promotions).
    *   Dùng để trả lời câu hỏi "Tại sao làm thế này?" hoặc "Làm thế nào?".

3.  **[Database Schema](Database_Schema.md)**
    *   Sơ đồ các bảng dữ liệu.
    *   Giải thích mối quan hệ giữa các bảng (Hóa đơn - Chi tiết, Sản phẩm - Danh mục).

4.  **[Viva Q&A (Câu hỏi vấn đáp)](Viva_Prep_QA.md)**
    *   Tuyển tập các câu hỏi thường gặp khi bảo vệ đồ án.
    *   Gợi ý câu trả lời ngắn gọn, súc tích và chính xác.

## Cách sử dụng
*   Đọc **Viva Q&A** trước để nắm các ý chính.
*   Đọc **Key Features** để hiểu sâu logic code nếu bị hỏi khó.
*   Xem **Architecture** để vẽ lại sơ đồ bảng/lớp nếu được yêu cầu.
